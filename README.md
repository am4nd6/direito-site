# Sistema Escritório de Advocacia

Este projeto é um sistema full-stack para um escritório de advocacia, organizado de forma modular para garantir escalabilidade e fácil manutenção.

## 🏗️ Arquitetura do Repositório

O projeto utiliza uma estrutura de pastas separada por responsabilidades:

### 🎨 [Frontend](./frontend)
Contém toda a interface do usuário (UI) e experiência do usuário (UX).
- **Framework**: React + Vite + TypeScript
- **Estilização**: Tailwind CSS + shadcn/ui
- **Scripts**: `npm run dev` para iniciar o desenvolvimento.

### ⚙️ [Backend](./backend)
Contém a API, lógica de negócio e integração com banco de dados.
- **Ambiente**: Node.js + TypeScript
- **Estrutura**: Organizado em Controllers, Services e Repositories.
- **Scripts**: (A definir conforme o framework escolhido)

---

## 🚀 Como começar

### Pré-requisitos
Certifique-se de ter o **Node.js (v18+)** instalado em sua máquina.

### Executando o Frontend
1. Acesse a pasta: `cd frontend`
2. Instale as dependências: `npm install`
3. Rode o projeto: `npm run dev`
